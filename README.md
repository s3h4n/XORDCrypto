# XORDCrypto 🔑

XORDCrypto is a simple python cli tool that can encrypt and decrypt files using XOR encryption.

## Installation ✨

Clone this repository.

```bash
git clone https://github.com/s3h4n/XORDCrypto.git
```

## Usage 🔥

```python
python3 -m XORDCrypto install
```
